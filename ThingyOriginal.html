Conservation International

Intoduction

This website is made by four beginner coding students that learn how to code in less than a month. We have come together with a common goal this summer to learn how to code and raise awareness of global warming in order to live a healthier and more productive life for everyone. We carefully picked an organization called 'Conservation International' which has the same goal as we do. With this organization we hope to raise money to support a cause to protect the most essential things that the earth offers us.
